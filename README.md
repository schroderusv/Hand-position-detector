# Hand-position-detector
The project is executed with ti sensortag. The device must be in the users hand while committing the movement.
There is 3 different movements the device can detect; High five, waving, and and maraca playing. It also sends the information about the movement to local similar devices and receives other device's movements. It contains additional features; if the movement is detected, the device plays a happy sounding ring and blinks a green led light. If not, the device plays a sad ring and blinks a red led light.
